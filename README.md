# 社区投稿AI处理助手 - 使用说明

## 📋 工具简介

本工具利用AI技术帮助社区更快处理投稿，自动完成：

- **自动分类**：将事件分为教育、劳动、校园其他等类别
- **智能摘要**：把长投稿浓缩成3-5句话关键信息
- **影响评估**：给出热度评分（1-10分）及评估理由

## 🚀 快速开始

### 1. 启动工具

打开命令提示符（CMD）或PowerShell，进入项目目录：

```bash
cd 你的保存的项目地址
```

安装依赖（首次使用）：

```bash
pip install -r requirements.txt
```

启动应用：

```bash
python main.py
```

启动后会显示一个本地URL（如 `http://localhost:7860`），复制到浏览器打开即可使用。

### 2. 使用步骤

1. **输入内容**：将投稿内容粘贴到左侧的"事件描述"文本框中
   - 支持复制微信、小红书、抖音等平台的内容
   - 建议内容不少于50字，以保证分析效果
2. **开始分析**：点击"开始分析"按钮
   - 系统会自动调用AI进行分类、摘要和评估
   - 分析过程大约需要10-30秒，请耐心等待
3. **查看结果**：
   - "分析结果"标签页：查看事件标题、类型、核心摘要和影响评估
   - "完整报告"标签页：查看格式化的Markdown报告
4. **导出报告**：
   - 点击"下载TXT报告"：下载纯文本格式报告
   - 点击"下载Word报告"：下载Word格式报告

## 📊 事件分类说明

工具会自动将事件分为以下类别：

| 类别   | 说明                |
| ---- | ----------------- |
| 教育   | 涉及学校教育、培训、学生相关问题  |
| 劳动   | 涉及劳动纠纷、工资待遇、工作环境等 |
| 校园其他 | 校园内非教育类问题         |
| 社区服务 | 社区设施维护、物业服务等      |
| 公共安全 | 治安、消防、交通安全等       |
| 环境卫生 | 垃圾处理、噪音污染、卫生问题等   |
| 邻里纠纷 | 邻里之间的矛盾和冲突        |
| 投诉建议 | 对政府或机构的投诉和建议      |
| 其他   | 不属于以上类别的事件        |

## 🔥 热度评分标准

热度评分范围为1-10分：

- **1-3分**：影响范围小，传播性低
- **4-6分**：中等影响，有一定传播潜力
- **7-10分**：高热度事件，可能引发广泛关注

## 💡 使用提示

1. **内容质量**：投稿内容越详细，分析结果越准确
2. **隐私保护**：请勿输入个人敏感信息（如身份证号、手机号等）
3. **网络要求**：需要联网使用，确保网络连接稳定
4. **报告保存**：建议及时下载保存分析报告，刷新页面后会丢失

## ❓ 常见问题

**Q：为什么分析结果为空？**
A：请检查网络连接，或确认输入内容不为空。

**Q：分类结果不准确怎么办？**
A：可以尝试增加描述内容的详细程度，或重新提交分析。

**Q：报告下载失败？**
A：请检查浏览器下载权限设置，或尝试更换浏览器。
